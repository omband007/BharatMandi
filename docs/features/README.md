# Features Documentation

Documentation for all Bharat Mandi features.

## Available Features

### 🎙️ [Kisan Mitra](kisan-mitra/)
AI-powered voice assistant for farmers
- Multi-language support (11 Indian languages)
- Voice input and output
- Conversational AI with AWS Lex
- Crop prices, weather, farming advice

### 🌐 [Translation](translation/)
Internationalization and translation services
- 11 Indian languages supported
- AWS Translate integration
- Batch translation optimization
- Translation feedback system

### 🏆 [Grading](grading/)
AI-powered produce quality grading
- Computer vision analysis
- Crop type detection
- Quality grades (A/B/C)
- Digital certificates

### 🛒 [Marketplace](marketplace/)
Digital marketplace for agricultural produce
- Listing management
- Media support (photos, videos)
- Transaction workflow
- Escrow payments

### 🔐 [Authentication](authentication/)
User authentication and authorization
- JWT-based authentication
- Role-based access control (RBAC)
- User management

### 🌾 [Crop Detection](crop-detection/)
AI-powered crop identification
- Image-based crop detection
- Hugging Face Vision Transformer
- Support for common Indian crops
